# Climber's Corner — TFCC Recovery & Gym Reviews

This is a beginner-friendly static website. You can edit it with any code editor and deploy it for free.

## Pages

- `index.html` — homepage
- `tfcc.html` — TFCC recovery guide
- `videos.html` — curated exercise video links
- `gyms.html` — climbing gym reviews and review template
- `about.html` — project/about page
- `styles.css` — all visual design
- `script.js` — mobile navigation toggle

## How to edit

Open the folder in VS Code. Edit the text inside each `.html` file.

To add a new gym review, copy this block from `gyms.html`:

```html
<article class="review-card">
  ...
</article>
```

Then paste it below the existing reviews and change the gym name, location, and review text.

## How to preview locally

Double-click `index.html`, or use VS Code Live Server.

## How to publish

### Option 1: GitHub Pages
1. Create a GitHub repository.
2. Upload these files.
3. Go to Settings → Pages.
4. Select your main branch and root folder.
5. Publish.

### Option 2: Netlify
1. Go to Netlify.
2. Drag this whole folder into Netlify Drop.
3. Netlify will publish it automatically.
