About this project

Built for all climbers.

Climber's Corner is a website for searching climbing gyms, reviews, and training exercises!

Our values

Clear basic information

I want basic gym information to be easy to find: location, address, opening hours, climbing type, contact details, and map links. Reviews should be personal, but basic information should be organized and practical.

Medical disclaimer

This site is not medical advice. It does not diagnose, treat, or prescribe. Consult a qualified clinician before starting rehabilitation exercises, especially if pain followed a fall, symptoms are severe, or the wrist feels unstable.
